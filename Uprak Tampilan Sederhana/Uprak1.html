A single-page application (SPA) is a web application or web site that interacts with the user by dynamically rewriting the current page rather than loading entire new pages from a server. This approach avoids interruption of theuser experience between successive pages, making the application behave more like a desktop application

The origins of the term single-page application are unclear though the eoneep concept was discussed at least as early as 2003.
